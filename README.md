# 🌷 A Pastel Garden for Luckyyyy!

A bespoke, atmospheric web experience crafted with a soft **"Spring-Pastel" aesthetic**. This project features floating bubble physics, interactive particle systems, and a nostalgic Polaroid-style reveal.

## ✨ Key Features

### 🌌 Atmospheric Engines
* **Soft Floating Bubbles:** Large, translucent gradient orbs that drift across the screen to create depth.
* **Fireflies & Stars:** A mix of pink and lilac particles that twinkle and wander using organic sine-wave math.
* **Shooting Stars:** Periodic high-speed animations that bring a sense of magic to the night sky.

### 🖱️ Micro-Interactions
* **Fluid Cursor:** A pink-tinted cursor with a smooth trailing effect for desktop immersion.
* **Emoji Sparks:** Interactive 🌷, ✨, and 🌸 bursts that spawn at your fingertips on every click.
* **Floral Petal Rain:** A celebratory waterfall of tulips, sunflowers, and hearts triggered upon entry.

### 🎵 Audio Synthesis
* **Web Audio API:** Generates a custom high-pitched melodic chime sequence in real-time upon entry—no external MP3 files required.
